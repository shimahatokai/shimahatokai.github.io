教锤子科技做设计

正如每个人都可以教苹果的设计师比如 Jonathan Ive 做设计一样，每个人也可以教锤子科技做设计，因为目前世界上的手机都不尽如人意，每个人可能都会梦想有一部完美的手机，dream phone。其他手机就算了，但是锤子科技不一样，他是一家以美就是生产力的设计驱动型公司。

我们的使命是用完美主义的工匠精神，打造用户体验一流的数码消费类产品（智能手机为主），改善人们的生活质量。

既然是完美主义，一流的用户体验，改善人们的生活质量，我们希望名实相符。他从苹果学到很多，我们希望他精益求精。更何况他出过迄今世界上唯一一款值得买的智能手机，坚果 U1，虽然问题也很多。

硬件

开机键

smartisan T1 的开机键和耳机孔是对称的，这是锤子科技一贯主张的对称美学。T2 取消开机键，开机键和 Home 键结合，长按 Home 键开机。

M1 取消开机键，开机键和 Home 键结合，长按 Home 键开机。

坚果 U1 开机键为长条形，顶部设计和 iPhone 4 类似。坚果 pro 开机键改在右侧。锤子科技一贯主张的对称美学，没有了。

开机键和 Home 键结合不行吗？当然行，但是我们需要一个单独的开机键，按下开机键，屏幕亮了。就像我们按下灯的开关，有光；我们打开电风扇，来风；我们拧下水龙头，出水。这一切都是人的自然直觉。如果按下一个键，它既可能是开机也可能是回到主页，既会引起用户困惑，也可能会按错了，误触发。它似乎无关设计，它是一种心理需要，通俗一点，叫情怀。这是为什么 smartisan os 要拟物，这为什么深泽直人的 cd 机要像老式的壁挂式风扇一样设计成下拉播放。这是人性，让我们感到手中的这件东西并不是死物，它是有温度的。

3.5mm 耳机孔

T1 的耳机孔和开机键对称，T2 顶部只有一个耳机孔。M1 耳机孔在底部。坚果 U1 耳机孔在顶部，坚果 pro 取消耳机孔，附送一根3.5mm 耳机转接线。

耳机孔应该在顶部还是底部，这是老罗纠结过的哲学问题。它和手机是否放入口袋有关。如果手机顶部有耳机孔和开机键，那么手机可以底部朝下，顺势放入口袋，这是人的自然反应，这也是 iPhone 4 的设计。如果像 iPhone 5 一样顶部有开机键，耳机孔却在底部，手机只能顶部朝下放入口袋，这极易碰到开机键，引起误触发。

现在大部分的耳机都是3.5mm 的插口，而坚果 pro 却取消耳机孔，如果不用转接线，还得另买耳机，这实在是不人性。用了转接线，既不优雅又会占用充电接口，还容易损坏，万一要充电，无法充电。有人说可以用蓝牙耳机嘛，但是蓝牙耳机首先不优雅，其次耗电，还有信号不好。

距离/光线传感器

细心观察的人会发现，坚果 pro 的距离/光线传感器的开孔比坚果 U1 大了不少。坚果 pro 宽度（宽73.4 mm）比坚果 U1（宽76.9 mm）小，开孔却大了，比例失衡，整体不和谐。用老罗的话说，这就不够优雅了，不够精致了。用许岑的话说，这就不高贵了。

双侧键

坚果 U1 的双侧键太长，或许是因为5.5寸的大屏，为了整体和谐，但其实和开机键长度一样就好了。

传统的截屏是按开机键 + 音量键或 Home 键，一定得双手操作，还显得笨拙无比。但是 T1，U1 ，T2，M1都有双侧键，按住截屏，长按滚动截屏，单手操作，想截就截，异常方便。不知道坚果 pro 为什么要取消，为了降低工艺难度牺牲用户体验？

虚拟按键

T1，T2 都有3个实体按键，坚果 U1 为3个虚拟按键，M1 和坚果 pro改为一个圆形 Home 键。

T1，T2 实体按键是为了防止传统安卓三大按键的误触发，实体按键如果放在口袋里，同样容易误按，还增加损坏的机率。为了解决误触发，后来的安卓版本干脆把三大键设置在屏幕内，完全是画蛇添足。三大键的功能是返回，主页，菜单，后来 M1 和坚果 pro 的一个圆形 Home 键就能实现这三个功能，但是仍然无法解决误按的问题。其实只要一个类似 iPhone 的悬浮球就好了，轻按一下返回，双击清理后台，长按返回主页，下拉是快捷设置菜单。（坚果 U1 要实现此功能只需在代码里改三个数字或者下个 app 屏蔽三个虚拟键，再下一个悬浮手势操作 app 即可。）如果不介意抄 iOS，甚至连这个悬浮球都可以不要，只要设计成在屏幕上左滑返回，下拉是通知和搜索，从底部上滑是快捷设置菜单和后台程序。

但是悬浮球出现在屏幕上，影响整体和谐。如果不用时自动隐藏到侧边，要用时呼出，多好。但是这样仍会增加操作步骤，破坏用户体验。如果这个悬浮球在屏幕外就好了，这就是魅族 MX3 最初的小圆圈。这样既能解决误触发，又不需要那么多实体键。这也是为什么一定要保留一个开机键。

指纹识别

M1，坚果 pro 都有前置指纹识别，而且是圆形的。前置和圆形是符合人直觉的。这也是为什么 iPhone 的 Home 键是圆形并且中间有个灰色的小方框。因为人看到手机上的这个 Home 键，虽然它实际和周围面板齐平，但视觉效果像平面上有一个凹陷，手指会自然放上去按压。

可问题是我们现在真的需要指纹识别吗？指纹用于代替密码，用于支付或者确认，但我们真的记忆如此之差，连6个数字都记不住吗？

Micro/Type-C USB 接口

锤子系列手机的数据线有所谓的凹陷防呆设计，这是因为 Micro USB 接口有正反，插反了会损坏接口。其实这是一个伪问题，如果接口没有正反之分，根本就不需要这个防呆设计。所以苹果发明 lightning 接口，正插反插都没有问题。现在业界都在转向 Type-C USB 接口，正插反插都没有问题，但是手机上接口内仍然有一块金属触板，损坏的机率远比 lightning 接口髙。

有心人可能会注意到，坚果 U1 数据线插口一般在一年到一年半就坏，如果按每天充一次电（或者每天拔插一次）计算，相当于只能拔插500次左右。很多人的原装数据线坏了，手机没有坏，但是他们以为是手机坏了，结果换新手机。其实换一根数据线就好了。为什么手机没坏，手机上Micro USB 接口也没坏，但是数据线坏了不能充电？怎么其他厂商的数据没出现这种问题？这当然是设计的问题，工程机做好都会有使用寿命测试，按键，电池，不知道是不是忽略数据线的插口测试。

机身尺寸

二者虽然都是5.5寸，但是坚果 pro （高155 mm）比 坚果 U1 （高152.9 mm）更长，这导致坚果 pro （宽73.4 mm）虽然宽度比 坚果 U1（宽76.9 mm）更修身，但看起来更大。虽然坚果 pro （厚 6.98 mm）比坚果 U1 （厚9.0 mm）更薄，但看起来更厚。这也是用户体验的一种。

这些都不重要，重要的是5.5寸。老罗曾经信誓旦旦的说绝不做3.5寸以上的手机，因为乔布斯说3.5寸是单手操作的黄金比例。结果 T1 出来就是5寸，5寸还能勉强单手。至于随后的 5.5寸的 U1，5.2寸的 M1，5.7寸的 M1L，虽然老罗想了很多软件上的办法提升用户体验，但还是不如人意。单手操作是一种人民群众喜闻乐见的良好用户体验，为什么要竟视若无睹呢？

全面屏

2017年，似乎各大厂商都忽然集体进入全面屏时代。真正的全面屏当然是正面只有一块屏幕，没有任何开孔。做不到的厂商则使尽浑身解数尽量缩小上额和下巴，恨就恨代工厂的技术不给力。做成全面屏，三大虚拟键必然放到屏幕内或者用悬浮球手势操作代替。三大虚拟键放在屏幕内会占屏幕一部分位置，相当于有一个内置的下巴，只不过以前是在屏幕外，现在移到屏幕内。且不说和悬浮球一样影响界面整体和谐，破坏用户体验，全面屏根本就没有想过一个足够宽的手机下巴存在的意义。

一个足够宽的手机下巴存在的意义是什么呢？是一种手机用户自然的手指摆放姿势需要，更是一种心理需要。我们握住手机，拇指自然地放在手机下方——下巴处，这是我们的正常反应。有一个下巴，我们知道拇指该放在哪，我们觉得自然，舒适，安心。一旦取消下巴，我们自然的握机姿势还是会把拇指放在下方，但是没有一个足够宽的下巴给我们一个范围，一个舒适的心理范围，会有种手足无措的挫败感，让人沮丧。这样的用户体验极差。做手机设计或者产品经理，不是技术过硬，有多年从业经验的理科生就可以的，有时还需要知道点心理学。即使不是心理学家，也应该知道点人性，如果他的思想还没有因为专业的理工科教育变得机械的话。其实本来就不应该存在文科生，理科生。正常的教育，是人性的教育，是全面发展的教育。强行的分科，造成大部分文科生缺乏科学常识，怪力乱神。大部分理科生思想僵化，灭绝人性。

软件

smartisan os 是锤子科技的核心科技，是锤子科技注重用户体验的体现，但是设计并不是那么完美。

开机界面

按开机键点亮屏幕后，顶部居中有一个时间，底部也有一个时间。但是人的视觉往往是先从往下看的，顶部看到一个时间，底部又看到一个时间，似乎有点多余，只要日期就好了。

每次关机后再重新开机，等锤子 logo 闪烁完，在进入开机界面前，总会来一段 starting apps …，然后漫长等待，体验实在是不好，还不如一直闪烁锤子 logo 也好。

顶部状态栏

顶部状态栏左边如果只有一张卡，会显示 China Mobile，如果有通知 China Mobile 会隐藏。因为 smartisan os 手机和 wifi 信号状态都是放在右边，而不是像 iPhone 一样放在左边。或许是为了对称美学，避免左边空旷不和谐，所以才设计成显示 China Mobile。其实是没必要的，因为右边也有手机信号状态，再左边显示 China Mobile，有点多此一举。实不知空旷反而更容易让用户注意到有消息通知，这也是关注用户体验。

时钟

锤子时钟的图标是拟物的，但指针却不动。既然要拟物，为什么不能像 iPhone 那样做成动态的呢？只要加几行代码就够了。

日历

锤子日历的图标是动态的，但是只有日，没有星期。大多数人除了看日期，也想知道星期，尤其是上班族，为什么不能一目了然呢？

音乐播放器

最初的 smartisan os 音乐播放器 Cloud （云音乐）没有 Recommend（推荐），只有单纯的搜歌功能，这是极好的，想听什么歌，自己找就好了，一个真正的文艺青年，难道还不知道自己要听什么吗？可能大部分人只是想要一个单纯的音乐播放器。开始现在（截至 smartisan os v3.7.6.1）音乐播放器的底部导航栏还加入 Radio，Albums，Songs，还有太多放不下，还加了一个 More。Albums 和 More 的内容都可以合并到 Songs 里，Radio 如果实在要，可以合并到 Cloud 里，因为不联网，也用不了。

浏览器

锤子阅读号称是信息过载年代的干净阅读，但是锤子浏览器却不干净。一个浏览器，只需要能搜索，可上网就行了。锤子浏览器却有天气（不是已有锤子天气了么？），热门网站链接（删都无法删），自家广告（比如锤子阅读），各种网站精选的八卦。锤子应用商店有一款大小只有二三百 K 的浏览器，比自带的锤子浏览器几十 MB 的不知道干净到哪里去了，下载安装量比所有锤子手机总销量还多，劲大，好使，自从有了它，咱再也不用锤子浏览器。

游戏中心

对不玩游戏的人来说，游戏中心实在是多余的，但又不能卸载，实在是不人性。

搜索

smartisan os 已经有底部上滑全局搜索，不知道为什么还要另外再做一个搜索。完全用不到，实在是多余。

安全中心 （Phone Manager）

安全中心的一些功能本来应该是在设置里的，比如 Permission Manager，Mobile Data Management，Storage Management，Advanced Settings。做一个独立的安全中心，功能却和设置重叠。

远程协助

创意确实不错，但是用到的机会并不多，有需要装一个就好了，但是不需要的却不能卸载。

One Step/Big Bang/闪念胶囊

这是 smartisan os 3.0 主推的三个功能，但是使用机会实在是有限。

One Step 要从屏幕顶部左上角或者右上角触发，但是要双手操作，极容易误触然后退出，更不用说像坚果 U1 这样底部还有三个虚拟键的。

Big Bang 复制搜索确实不错，但是拇指长按屏幕既可能触发 Big Bang，也可能触发其他应用自带的长按屏幕触发相关功能，比如微信里的选择复制。Big Bang 似乎还不能解决和其他应用长按触发功能冲突的问题。

闪念胶囊其实是 录音机+语音识别，有创意，但是看到屏幕上一条条五颜六色的弹幕一样的胶囊，用户体验实在是不好，尤其有强迫症的，更不用说耗电，占内存。真的有这种需要的人，手机自带的录音机或许就够了。

预装第三方应用

老罗发布会上一直强调锤子手机可以卸载全部预装第三方应用，确实是可以卸载，但只是表面上卸载，第三方应用的 apk 安装包还是留在系统内存里占用内存而且无法删除，如果恢复出厂设置，第三方应用又回来了。如果不介意占的这点内存，也没什么好说，但是强迫症怎么办？锤子科技不是以完美主义自居，专著治疗强迫症吗？

one more little thing

坚果 U1 会不时莫名自动重启，向锤子售后反映一年多，至今还是无法解决，或许只是个例。

好的手机是怎样的

好的手机应该尽量延长使用寿命，而不是用个半载一年，电池就报废了，接口就坏了，按键就掉了，只能丢掉了。就算是回收，也应该易于回收再循环，像供应链一样有完善的回收链。一个公司除了赚钱，还有社会责任的，with great power comes great responsiblity，智能手机厂商其实是地球上最大的污染来源。手机硬件更新换代，手机系统升级，大量的手机用个一两年就不能用，因为再用就太卡了，用户体验太差了。就算坚持使用，电池也损耗不行了，大部分手机电池的设计使用寿命是5年，实际可能只有1年，新机两天一充，现在一天充五次。即使厂商继续维护，用户自己也要淘汰了。NOKIA 时代一个手机用好几年的情况，很难再见到了。这些淘汰的手机如果用户不自己收藏，就只有丢掉或是回收。这些回收的手机去了哪里呢？看看贵屿，还有更多的类似的地方。我想没人会乐意生活在那样的地方，更不愿看到自己的后代生活在那样的地方。

好的手机还应该是人性的，设计是自然天成的，操作符合人的直觉。讲用户体验，做产品经理，比起计算机科学与技术专业的毕业生，似乎需要更心理学家，但是如果我们的天性没有被专业的教育抹杀，我们的审美没有被潮流扭曲，每个人天生都是最好的产品经理。

smartisan T1 曾经给我们很多惊喜，我们相信罗老师至今没有忘记做手机的初心，我们期待 smartisan T3 再给我们惊喜。
