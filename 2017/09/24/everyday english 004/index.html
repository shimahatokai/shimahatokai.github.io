日常英语談笑風生 004

求人也可以求得高贵，不卑不亢，不需要低声下气。因为大家都是平等的，别人没帮，不必耿耿于怀；帮了人，也不要斤斤计较，觉得对方就欠了自己一个人情。帮，是因为我们高贵，举手之劳，手有余香。不帮，也不必自责，没有什么是理所当然的，帮人要花时间，精力，财力，物力。有的人有能力帮，有的人没能力帮，何况还有帮倒忙的。有人觉得，我这次帮了你，下次我有事，你也得帮忙，不然就是不知道知恩图报。把助人为乐当成交易，这，不高贵。

求，中文虽然一个字，英语却有不同（其实中文也有不同，比如 ‘央’）。中国学生常常混淆两个单词：request，require。

request，OALD 解释为：ask for something or ask somebody to do something in a polite or formal way。有中文 ‘希望’ 的意思，不是强制的，有点像中文的 ‘劳驾…，能否屈尊…，能否赏光…，可以赏脸…吗’，但又没有中文，不经意间还流露出一丝高贵，因为这个词本身就是高贵的（formal），一般人也就用用 Could you…，May I…。

require，OALD 解释为：make somebody do or have something, especially because it is necessary according to a particular law or set of rules。有中文 ‘必须’ 的意思，因为是法律或规定要求如此。像米国大学研究院的入学要求就是 requirement，比如其中有一条，TOEFL iBT不低于 108。

邀请也是求人的一种。

求人：劳驾…

Can you / Will you

Will you 有点单刀直入，尤其是要老友帮点举手之劳的事，类似中文的 ‘客气话就不说了…’。

Could you / Would you / Do you think you could

客气一点。求的人可能只是点头之交，做的事可能要费点心。

Would you mind doing something / I wonder if you would mind doing something

The more, the better. 更客气一点。如果不介意，可以说 No / Of course not。如果不愿意，可以说 I’m afraid / I’m sorry…。

Could you possibly / Is there any way you could

很可能要给人添麻烦了。

Can I ask a favour / Could you do me a favour / Could I ask you to do me a favour

高贵。如果人家答应了，别忘了说 ‘I promise I’ll return the favour / Thanks, I really owe you one.’

I was wondering if you could / Would it be possible for you to do

高贵。

I would be grateful if you could / I would appreciate it if you could

商函公文。

Would you be so kind as to do

央倩素未谋面的人，要多客气有多客气。

We request that … / You are requested not to do something

‘烦请…’，‘用户须知’，用在指示和公告。

贴心提醒：任何时候不要忘了加上一个 please。

有时求人不如求己，为表尊重，先征求一下别人的同意，把上面求的 you 改成 I / me。

  • Can I / Is it all right if I / Is it OK if I

  • May I / Could I / Do you think I could /

  • Do / Would you mind if I did / Would I be able to

  • I was wondering if I could / I was wondering if it might be possible for me to do

  • Would it bother you if I did / Would it be OK if I did

    注意，虚拟语气，I 后面跟动词过去式。

  • With your permission, I’d like to do

邀请：能否赏光…

Are you free on Saturday

凡事预约，方显高贵。‘请问周六有空吗？’如果佳人有约，就可以说 ‘I’ve got to meet someone then. Maybe another time?’

I would like to book…

用于酒店订房，餐厅留位，预约门诊…

Do you fancy something

地气。英国人说。‘要试下吗？’，超市试吃可用。

Are you up for something / Do you want to meet up for something

地气。英国人说。‘下班后去喝一杯（搓一顿）吗？’不想去，就说 I’d love to but I’m really busy，‘姐还要加班呢。’

Do you feel like something

地气。英国人说。‘来一串吗’。

Would you like something / Would you like to do something

高贵。如果是 ‘对不起，我已经有男朋友了’，可以地气一点说 ‘I really appreciate the invite, but sorry – I can’t make it. ’

如果怕拒绝，也可以先提议一下，试试口风，再见机行事。比如：

  • Why don’t you / we

  • Let’s

  • If I were you, I would

  • How / What about

  • What do you think about

  • What if / suppose

  • May / Might / Maybe we could / Perhaps you could

  • You might like to / You may want to
